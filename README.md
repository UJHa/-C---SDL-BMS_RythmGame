# -C-SDL-BMS_RythmGame
bms파싱하여 노트를 배치한 리듬게임


- 실행파일 다운로드 방법
  1. 실행파일.zip 다운로드 후 압축 해제
  2. 실행파일/Release/RythmGame.exe 실행


- 게임 플레이 키

  곡 선택 : 상하좌우, space키

  키 입력 : A,S,D,F,G,H,J,K
